# 吻合器不良事件跨数据库迁移学习与风险挖掘研究项目 (Stapler Research)

本专用文件夹用于承载《基于大语言模型的吻合器不良事件跨数据库迁移学习与风险信息挖掘研究》的所有开发脚本、中间数据以及研究报告。

---

## 📁 目录结构规划

* 📂 **`scripts/`**：存放本项目相关的全部开发脚本。
  * `export_stapler_db.py`：从大数据库中跨库复制并克隆吻合器专有分析库的主程序。
  * `verify_stapler_db.py`：对克隆后的数据库做物理索引和字段级数据一致性校验的工具。
  * `standardize_data.py`：用于第一步规范化源域（MAUDE）与目标域（南京）数据的脚本。
  * *(后续将新增)* `llm_alignment.py`：用于第二步大模型实体提取与 IMDRF 编码映射的脚本。
* 📂 **`data/`**：存放本项目特有的规范化、去噪后及中间态数据。
  * `source_standardized.pkl`：源域（MAUDE）清洗合并后的特征矩阵序列化文件。
  * `target_standardized.pkl`：目标域（南京）清洗净化后的特征视图序列化文件。
* 📂 **`reports/`**：存放本项目的学术设计规划、论文大纲及实验交付报告。
  * `thesis_design_plan.md`：本课题的详细研究方案与 7 章节毕业论文大纲。
  * `walkthrough.md`：专有分析库的数据迁移交付报告。
